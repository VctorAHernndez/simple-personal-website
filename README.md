# Simple Personal Website

Simple personal website made in React back in 2019 in order to test my skills in React, Sass, deployment pipelines and whatnot!

## Available Scripts

In the project directory, you can run:

- `npm start`
- `npm test`
- `npm run build`
- `npm run eject`
